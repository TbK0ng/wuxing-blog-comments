## 🔧 常见问题

### Q: SubAgent无响应怎么办？
A:
1. 检查配置格式是否正确
2. 验证工具权限设置
3. 确认网络连接正常
4. 尝试重新注册SubAgent

### Q: 如何优化SubAgent性能？
A:
1. 减少不必要的工具配置
2. 简化指令复杂度
3. 使用缓存机制
4. 合理分配任务

### Q: 多SubAgent协作如何避免冲突？
A:
1. 明确分工职责边界
2. 建立标准化接口
3. 实现版本控制
4. 设置质量检查点

### Q: 如何创建有效的自定义SubAgent？
A:
1. 明确定义专业角色
2. 提供详细工作指令
3. 配置合适的工具集
4. 包含实际使用示例
