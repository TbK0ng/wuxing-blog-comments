# ClaudeCode快捷键完全指南

## 🎯 基础导航快捷键

| 快捷键 | 功能 | 适用平台 | 说明 |
|--------|------|----------|------|
| `Ctrl+R` | 反向搜索历史命令 | Windows/Linux/macOS | 快速查找之前执行的命令 |
| `↑ / ↓` | 浏览输入历史 | Windows/Linux/macOS | 上下箭头查看命令历史 |
| `Tab` | 命令和文件路径补全 | Windows/Linux/macOS | 智能补全，提高输入效率 |
| `Ctrl+C` | 取消当前输入或中断生成 | Windows/Linux/macOS | 立即停止当前操作 |
| `Ctrl+D` | 退出当前会话 | Windows/Linux/macOS | 优雅退出ClaudeCode |

## 🖥️ 终端控制快捷键

| 快捷键 | 功能 | 适用平台 | 说明 |
|--------|------|----------|------|
| `Ctrl+L` | 清除终端屏幕 | Windows/Linux/macOS | 清理屏幕，保持界面整洁 |
| `Cmd/Ctrl + L` | 清屏 | macOS/Windows/Linux | 快速清屏的替代方案 |
| `Ctrl+U` | 删除整行 | Linux/macOS | 从光标位置删除到行首 |
| `Ctrl+K` | 删除到行尾 | Linux/macOS | 从光标位置删除到行末 |

## ⌨️ 输入控制快捷键

| 快捷键 | 功能 | 适用平台 | 说明 |
|--------|------|----------|------|
| `Option+Enter` | 换行 | macOS | 在不执行命令的情况下换行 |
| `Shift+Enter` | 换行 | Windows/Linux/macOS | 需要先运行`/terminal-setup` |
| `Cmd/Ctrl + R` | 详细输出 | macOS/Windows/Linux | 获取更详细的命令输出 |
| `Esc` | 中断当前操作 | Windows/Linux/macOS | 停止正在执行的任务 |
| `Esc + Esc` | 返回上一步 | Windows/Linux/macOS | 撤销上一次操作 |

## 🔧 搜索和导航

### 文件内搜索
```bash
# 快速搜索当前文件内容
Ctrl+F      # 向前搜索
Ctrl+G      # 跳转到指定行号
Ctrl+Shift+F # 全局搜索
```

### 历史命令导航
```bash
# 高效的历史命令操作
Ctrl+R      # 反向搜索历史
!!          # 执行上一条命令
!$          # 获取上一条命令的最后一个参数
Ctrl+S      # 前向搜索历史
```

## 🚀 快捷键组合技巧

### 开发常用组合
1. **快速测试循环**
   ```bash
   [编辑代码] + Ctrl+S + [运行测试命令]
   ```

2. **Git提交组合**
   ```bash
   git add . + Ctrl+R[找到commit命令] + Enter
   ```

3. **项目切换组合**
   ```bash
   Ctrl+D + cd [项目目录] + claude
   ```

### 效率提升组合
1. **上下文重置**
   ```bash
   Ctrl+L + /compact + /add-dir src
   ```

2. **快速审查**
   ```bash
   /diff + Ctrl+R[找到review命令] + Enter
   ```

## 🔍 快捷键故障排除

### 常见问题解决
1. **快捷键不生效**
   - 检查终端配置
   - 确认没有其他软件占用快捷键
   - 重启ClaudeCode

2. **某些快捷键冲突**
   - 修改系统或IDE快捷键设置
   - 使用替代快捷键组合
   - 配置ClaudeCode自定义快捷键

3. **Tab补全不工作**
   - 运行`/terminal-setup`
   - 检查shell配置
   - 确认路径正确
